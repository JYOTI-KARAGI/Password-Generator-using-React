Password Generator using React

A simple and customizable password generator built using **React.js** and **Tailwind CSS**. It allows users to generate secure passwords by adjusting the desired length and choosing whether to include numbers and special characters.

Features

- Generate secure passwords instantly
- Choose password length (8–25 characters)
- Include/exclude:
  - Numbers (`0–9`)
  - Special characters (`!@#$%^&*`)
- Copy generated password to clipboard
- Responsive and clean UI with Tailwind CSS


 Tech Stack

-  React.js
-  Tailwind CSS
-  useRef, useState, useEffect, useCallback
